# 写真の基礎

---

## 露出の三角形

写真の明るさを決める3つの要素。この3つのバランスで露出が決まる。

### シャッタースピード
- 速い（1/1000秒など）→ 動きを止める。明るさは暗くなる
- 遅い（1/30秒など）→ 動きがブレる。明るさは明るくなる
- スローシャッター＋三脚で、光の軌跡や水の流れを表現できる

### 絞り（F値）
- 開放（F1.8など小さい数値）→ 背景がボケる。光をたくさん取り込む
- 絞る（F8など大きい数値）→ 全体にピントが合う。光は少なくなる
- ポートレートは開放寄り、風景はF8前後が基本

### ISO感度
- 低い（ISO100など）→ ノイズが少なくきれい。暗い場所では不足する
- 高い（ISO3200など）→ 暗い場所でも撮れる。ノイズが増える
- なるべく低く保ちつつ、シャッタースピードと絞りで調整するのが基本

## 構図の基本

### 三分割法
画面を縦横3分割して、被写体を交点や線上に置く。
中央に置くより、自然なバランスになりやすい。

### 日の丸構図
被写体を中央に置く。シンプルで力強い印象。
主役をハッキリ見せたいときに使う。

### 対角線構図
斜めのラインを使って、奥行きやダイナミクスを出す。
道や橋、建物のラインを利用しやすい。

### 余白（ネガティブスペース）
被写体の周りにあえて空間を残す。
余白があることで、主役が際立つ。

### フレーミング
木の枝、窓、アーチなど、自然や建物を「額縁」として使う。
被写体を囲むことで視線を誘導できる。

## 光の読み方

### 光の方向

**順光**
正面から光が当たる。明るくきれいに写るが、立体感が出にくい。

**サイド光**
横から光が当たる。陰影が生まれ、立体感が出やすい。
質感を描写したいときに有効。

**逆光**
後ろから光が当たる。被写体の輪郭が光り、雰囲気が出る。
露出が難しいが、決まると印象的な一枚になる。

### 光の質

**ハード光**
晴れた日の直射日光など。くっきりした影が出る。
コントラストが強くなる。

**ソフト光**
曇り、日陰、窓からの拡散光など。やわらかい影。
ポートレートに向いている。

### 時間帯
- **ゴールデンアワー**（日の出・日没前後1時間）→ 暖かい色味、斜めの光、長い影
- **ブルーアワー**（日没後・日の出前）→ 青みがかった静けさのある光
- **曇り**→ 均一なソフト光。色が鮮やかに出やすい

## ピント・フォーカス

### フォーカスの種類

**シングルAF（AF-S）**
シャッターを半押ししたときにピントを固定する。
止まっている被写体に向いている。

**コンティニュアスAF（AF-C）**
動く被写体を追い続ける。
スポーツ・動物・子供など動きのある撮影に。

**マニュアルフォーカス（MF）**
自分でピントを合わせる。
オールドレンズ使用時や、AFが迷う場面で使う。

### ピントを合わせる場所
- 人物は **目（特に手前の目）** に合わせるのが基本
- 風景は **手前から奥までシャープ** にしたければ、絞り込んで撮る
- 被写体の一部だけを際立たせたいときは、ピントの位置で主役を決める

## 撮影モード

| モード | 読み | 特徴 |
|---|---|---|
| **P** | プログラムオート | カメラが自動設定。初心者向け |
| **A / Av** | 絞り優先 | 絞りを自分で決める。背景ボケのコントロールに |
| **S / Tv** | シャッター優先 | SSを自分で決める。動きの表現に |
| **M** | マニュアル | 全部自分で決める。完全コントロール |

ストリートや日常スナップは **Aモード** から始めると扱いやすい。

---

## カメラの基本操作フロー

```
① モードを選ぶ（AかMが基本）
② ISOを設定する（なるべく低く）
③ 絞りを決める（ボケ感・被写界深度）
④ シャッタースピードを確認する（手ブレしない速度か）
⑤ 構図を決めてフォーカスを合わせる
⑥ シャッターを切る
```

## よく使う用語まとめ

| 用語 | 意味 |
|---|---|
| 露出 | 写真の明るさ全体のこと |
| 被写界深度 | ピントが合って見える範囲の深さ |
| ボケ | ピントが外れてぼんやり写る表現 |
| パンフォーカス | 画面全体にピントが合っている状態 |
| 白飛び | 明るい部分が真っ白になって情報が消えること |
| 黒つぶれ | 暗い部分が真っ黒になって情報が消えること |
| RAW | 未現像のデータ。後から細かく編集できる |
| JPEG | カメラ内で現像済みの画像データ |
| セパレート | 被写体と背景を明確に分離すること |
| ハイライト | 写真の中で明るい部分 |
| シャドウ | 写真の中で暗い部分 |
