# 構図の基礎

どうも、僕です。
今日は構図の話をしようと思います。

写真を撮っていると、なんかしっくりこない一枚ってあるじゃないですか。
露出も合ってる。ピントも来てる。でもなんか違う。

その「なんか違う」の正体、だいたい構図だったりするんですよね。

---

## 構図って、何を決めることなのか

ざっくり言うと、

**画面の中に何をどこに置くか**

それだけです。

同じ場所で同じ被写体を撮っても、カメラをどこに向けるかで全然違う写真になる。
それが構図の面白さというか、怖さでもあるなと。

意識するのは主にこの辺り。

・被写体をどこに置くか
・視線をどこへ流すか
・余白をどう使うか
・奥行きをどう出すか

難しく考えなくていいです。
一個ずつ試していけばいい。

---

## まず覚えるなら、三分割法

画面を縦横3等分して、その交点に主役を置く。

```
┌───┬───┬───┐
│   │   │   │
├───┼───┼───┤
│   │ ★ │   │
├───┼───┼───┤
│   │   │   │
└───┴───┴───┘
```

「なんで中央に置かないの？」ってなりますよね。

中央に置くと、なんか証明写真っぽくなりがちなんですよね。
少しずらすだけで、ぐっと自然に見える。

カメラのグリッド表示をオンにしておくと、撮りながら確認できるので便利です。

---

## 対称構図

左右、または上下がシンメトリーになっている構図。

安定感とか、静けさとか、そういう印象を出したいときに向いてます。
お寺の門、水面への映り込み、廊下の奥に続く扉とか。

ピタッとハマると気持ちいい構図です。

---

## 対角線構図

被写体や線を、画面の対角線に沿って配置する。

対称と真逆で、動きとかスピード感とか緊張感みたいなものを出したいときに。
道路、川、建物のラインなど、斜めに走るものがあったら積極的に使ってみる感じですかね。

---

## フレーミング

木の枝、窓枠、アーチ、トンネルの入り口。

こういう「自然の額縁」を使って被写体を囲む構図。
絵画を覗いてるような奥行きが生まれて、印象がガッと変わります。

「この隙間から向こうを撮ったら面白いかも」という直感を大事に。

---

## 視線を引っ張る線

道、川、線路、フェンス。

こういう「線」を使って、見る人の目を被写体まで自然に引き込む。

```
╲
 ╲
  ╲
   ★ ← 気づいたらここに目が行く
```

一本道が画面の隅から奥に伸びていると、「その先を見たい」って感覚になりますよね。
あれです。

---

## 前景・中景・後景の三層構造

手前・中間・奥、の三層を意識するだけで奥行きが出る。

| 層 | 役割 |
|---|---|
| 前景（手前） | 奥行き感を演出する。ボケさせることが多い |
| 中景（中間） | 主役。ここに目が集まる |
| 後景（奥） | 雰囲気と世界観を補完する |

手前に何か入れるだけで、「ただの写真」から「世界観のある一枚」になる。
逆に言えば、手前が空っぽだと平べったい写真になりやすいんですよね。

---

## 余白の話

余白って、ムダに見えるんですよね。最初は。

でも余白があると被写体が際立つ。
静けさや孤独感みたいな空気も出せる。

「思ったより少し広めに余白を取る」を意識するだけで、写真の呼吸が変わる感じがします。

---

## やりがちな失敗と対処

**いつも真ん中に置いてしまう**
→ 三分割の交点を意識して、あえて外してみる

**水平線が傾いている**
→ グリッドをオンにして確認。傾きは後から直せる。意図的に傾けるのはダッチアングルという技法なので、狙ってやるなら全然アリ

**背景がごちゃごちゃ**
→ 角度を少し変えるだけでスッキリすることが多い。撮る前に背景を一度ちゃんと見る

**被写体が窮屈そう**
→ 少し引くか、被写体の向きの先に余白を作る

---

## 伝えたい印象に合わせて選ぶ

| こんな印象を出したい | 構図・テクニック |
|---|---|
| 安定感・静けさ | 対称構図・水平線・下重心 |
| 動き・ドラマ | 対角線・リーディングライン |
| 孤独・ミニマル | 広い余白・被写体を小さく |
| 奥行き・広がり | 三層構造・前景を入れる |
| 迫力・存在感 | ローアングル・フレーミング |

---

## まとめ

構図に絶対の正解はないんですよね。
でも法則を知っていると、意識的に破ることができる。
そこからが面白くなってくる、かなと。

まず一個だけ試すなら、三分割法から。
それだけで、いつもと違う一枚が撮れると思います。
